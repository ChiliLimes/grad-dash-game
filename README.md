# 🎓 Grad Dash

I have one of my closest friends graduating from Georgia Tech, and I wanted to do something special for his graduation party. So I built this game. Beyond proud of you Treyvon — this one's for you. 🐝

---

## 🎮 How to Play

- **Space / Click / Tap** to jump
- **Double tap** to double jump
- **ESC / P** to pause

Survive all 4 years of college and make it to graduation. Easier said than done.

## 🗺️ The Journey

- **Freshman Year** — 0 to 100m
- **Sophomore Year** — 100 to 200m
- **Junior Year** — 200 to 300m
- **Senior Year** — 300 to 400m
- **Graduation Finals** — 400 to 600m (the boss level)

## 🛠️ Built With

- HTML5 Canvas
- Vanilla JavaScript
- CSS3

## 🚀 Running Locally

1. Clone the repo
2. Open `index.html` in your browser
3. That's it — no installs, no build steps
