# 🎓 Grad Dash

A fun graduation-themed endless runner game where you navigate through 4 years of college and survive to graduation!

## 🎮 Play the Game

[Play Grad Dash Online](#) *(Add your GitHub Pages link here after deployment)*

## 📖 About

Grad Dash is a browser-based platformer game that takes you through the college experience:
- **Freshman Year** (0-100m) - Getting started
- **Sophomore Year** (100-200m) - Finding your rhythm  
- **Junior Year** (200-300m) - Things get serious
- **Senior Year** (300-400m) - Almost there!
- **Graduation Finals** (400-600m) - The final boss level!

## ✨ Features

- 🎓 Complete onboarding with high school diploma and Georgia Tech acceptance letter
- 👥 Choose from 3 unique characters (Treyvon, Brad, or Taki)
- 📱 Mobile-friendly with touch controls
- 🎨 Beautiful themed levels that change as you progress
- 💀 Hilarious fail messages when you don't make it
- 🏆 Challenging obstacles and boss level

## 🕹️ How to Play

- **Space / Click / Tap** - Jump
- **Double Jump** - Press jump again while in the air
- **ESC / P** - Pause game

Avoid obstacles, survive all 4 years, and make it to graduation!

## 🛠️ Technologies Used

- HTML5 Canvas
- JavaScript (Vanilla)
- CSS3

## 🚀 Running Locally

1. Clone this repository
2. Open `index.html` in your web browser
3. That's it! No build process needed.
